4. Media Query Transitions

Resize your browser window

This box changes color, width, and rotation based on screen size with smooth transitions.
/* Media query transitions */ .responsive-box { transition: all 0.5s ease; } @media (min-width: 768px) { .responsive-box { background: #2ecc71; width: 80%; } } @media (min-width: 1024px) { .responsive-box { background: #e74c3c; transform: rotate(-2deg); } }
